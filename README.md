# LearnMate AI

LearnMate AI is an interactive, AI-powered learning assistant built with React, Vite, and Google AI Studio. 

## Instant Deployment

You can instantly deploy this project to your own Google Cloud account using Cloud Run:

[![Run on Google Cloud](https://deploy.cloud.run/button.svg)](https://deploy.cloud.run/?git_repo=https://github.com/nishaworkactivity-wq/WarmupChallengeIntelligentAI.git)

Click the button above to automatically build and deploy the app to Google Cloud Run!

## Local Development

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file and add your `VITE_GEMINI_API_KEY`
4. Run `npm run dev`
